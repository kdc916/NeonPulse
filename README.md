🏃‍♂️ Neon Pulse (네온 펄스) - Cyberpunk Rhythm Runner

Neon Pulse는 네온사인과 사이버펑크 감성이 물씬 풍기는 웹 기반 끝없는 달리기(Endless Runner) 게임입니다. 화면을 가득 채우는 화려한 파티클 효과, 속도감 넘치는 플레이, 그리고 플레이어의 취향대로 꾸밀 수 있는 다채로운 커스터마이징을 제공합니다. 별도의 설치나 라이브러리 없이 웹 브라우저만 있으면 어디서든 즐길 수 있습니다.

✨ 주요 기능 (Key Features)

역동적인 파쿠르 액션: 기본 2단 점프를 지원하며, 점수가 3,000점을 돌파하면 숨겨진 '트리플 점프'가 해금됩니다. 선입력(버퍼링) 시스템을 적용해 조작의 답답함을 없앴습니다.

다양한 기믹과 아이템:

워프 포탈 (Gravity Portal): 중력을 반전시켜 천장을 달립니다.

비행 연료 (Flight Item): 무적 쉴드를 두르고 캐릭터 주변의 원형 게이지가 닳을 때까지 공중을 비행합니다.

대시 오브 (Dash Orb): 유리 장벽을 박살 내며 무적 상태로 돌진합니다.

사이버펑크 감성의 커스터마이징: 나만의 캐릭터(색상, 모양, 표정)를 만들 수 있습니다. 로컬 AI 알고리즘을 활용한 '사이버펑크 랜덤 닉네임 생성기'를 제공합니다.

화려한 시각 효과 (VFX) & 사운드: * 연속으로 별을 획득하면 발생하는 레트로 '글리치(Glitch)' 콤보 이펙트.

화면 셰이크, 파티클(먼지, 파편, 네온 고리 등), CRT 스캔라인 필터 적용.

Web Audio API를 활용한 자체 신디사이저(Synth) 사운드 효과.

시스템 코치 (System Coach): 게임 오버 시 사망 원인을 분석하여 팁과 위트 있는 피드백을 제공합니다.

반응형 UI (Responsive Design): 모바일(가로/세로) 및 PC 화면 비율에 완벽 대응하며, 모바일 기기의 노치 영역(Safe Area)과 주소창 잘림(100dvh)을 꼼꼼하게 처리했습니다.

🏗️ 아키텍처 및 코드 구조 (Architecture)

단일 index.html 파일로 구성되어 있지만, 유지보수와 확장을 위해 내부적으로 모듈화된 아키텍처를 채택했습니다.

StateFactory: 게임, 플레이어, 월드 상태를 관리하는 팩토리.

ObstacleFactory: 다양한 장애물과 아이템 객체를 생성.

LevelGenerator: 생존 시간에 비례해 난이도를 조절하며 일반/희귀 패턴 단위로 맵을 동적 생성 (절차적 생성).

PhysicsSystem: 중력 반전, 충돌 판정(히트박스), 아이템 상호작용 등 물리 로직 처리.

RenderSystem: Canvas 렌더링, 파럴랙스(Parallax) 스크롤링 배경, UI 및 파티클 드로잉 전담.

EffectSystem: 화면 진동, 파티클 이펙트, 플로팅 텍스트 관리.

UIService: DOM 엘리먼트 접근 및 UI 상태 업데이트 전담.

AudioSystem: Web Audio API 기반 효과음 생성기.
